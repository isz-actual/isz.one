# Make.com Workflows

This folder contains exported JSON configurations for all active Make.com scenarios.

## How to Export a Scenario

1. Open the scenario in Make.com
2. Click the three-dot menu in the upper right
3. Select Export Blueprint
4. Save the .json file here with a descriptive name

## Naming Convention

[brand]-[trigger]-[action].json

Examples:
- isz-order-confirm-notification.json
- northern-signal-new-subscriber-welcome.json
- reed-group-contact-form-hubspot-sync.json

## Recovery

To restore a scenario, import the JSON file into Make.com via Scenarios > Import Blueprint.
