# Prompt Library

Refined prompts used across Claude and other AI tools. Version-controlled so improvements are tracked and regressions are recoverable.

## Categories

- linkedin/ — Post generation, engagement replies, outreach sequences
- newsletter/ — Northern Signal issue drafts, subject line generation
- book/ — Outreach templates, review requests, positioning language
- research/ — Competitive intelligence, threat analysis frameworks
- isz/ — Product descriptions, campaign copy

## Format

Each prompt file should include:
- Purpose (one line)
- Model tested on
- Last updated date
- The prompt itself
- Any known limitations
